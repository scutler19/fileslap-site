Basic Example

import fetch from "node-fetch";
import fs from "fs";

const API_KEY = "YOUR_API_KEY";
const html = "<h1>Hello World</h1><p>This is a PDF generated by FileSlap.</p>";

async function convertHtmlToPdf() {
  const res = await fetch("https://api.fileslap.com/api/convert", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "X-API-KEY": API_KEY
    },
    body: JSON.stringify({ html })
  });

  if (!res.ok) {
    throw new Error(`HTTP ${res.status}: ${await res.text()}`);
  }

  const buffer = await res.arrayBuffer();
  fs.writeFileSync("output.pdf", Buffer.from(buffer));
  console.log("PDF saved as output.pdf");
}

convertHtmlToPdf();

Handling the PDF Response

- The API returns the PDF as a binary response.
- Save it to disk, send as an email attachment, or serve it in your app.

Error Handling

- 400: Invalid HTML or missing fields
- 401: Invalid API key
- 429: Rate limit exceeded (free plan: 5/day, 50/month)
- 500: Service unavailable

Handle errors with try/catch and check the response status.

Advanced Usage

- Use external CSS, images, or fonts in your HTML.
- Generate dynamic documents (invoices, reports, etc.).
- Integrate with Express, serverless functions, or background jobs.

Best Practices

- Keep HTML simple for best performance.
- Inline critical CSS for consistent rendering.
- Test with real data and edge cases.
